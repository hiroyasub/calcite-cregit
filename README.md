linq4j
======

A port of LINQ (Language-Integrated Query) to Java.

Download
========

    $ mkdir linq4j
    $ cd linq4j
    $ git init
    $ git remote add origin git@github.com:julianhyde/linq4j.git
    $ git pull
    $ git checkout master

Build and test
==============

    $ mvn compile
    $ mvn test
